# Format: dataset_name|cause_start cause_end|effect_start effect_end|weight|threshold_variable|threshold_value|regime_low_direction|regime_high_direction
# Directions: 1 = X->Y (cause->effect), -1 = Y->X (effect->cause)
# Regime Low: values <= threshold_value, Regime High: values > threshold_value
#
# Example: auto_mpg_horsepower|1 1|2 2|1|horsepower|140.0|-1|1

#        Above 140 HP: HP->MPG (engineering-driven: power drives consumption, direction=1)
